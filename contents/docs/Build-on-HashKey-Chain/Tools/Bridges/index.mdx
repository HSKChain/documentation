# Bridges

## Overview

Bridges are essential infrastructure components that facilitate the secure transfer of assets and data between HSKChain and other networks like Ethereum. They enable interoperability between blockchains, allowing decentralized applications (dApps) to leverage the advantages of multiple networks.

## Available Bridges

The following bridges provide secure and efficient connectivity between HSKChain and Ethereum:

### Superbridge

[Superbridge](https://bridge.hashkeychain.net) enables secure transfer of HSK and other supported assets from Ethereum mainnet (L1) directly to HSKChain.

**Testnet Environment:** [HSKChain Testnet Bridge](https://testnet-bridge.hashkeychain.net/)

### Orbiter Finance

[Orbiter Finance](https://www.orbiter.finance/en?src_chain=177\&tgt_chain=1\&src_token=ETH) provides trustless cross-rollup transactions of Ethereum and Bitcoin native assets in a seamless manner.

**Testnet Environment:** [HSKChain Testnet Bridge](https://test.orbiter.finance/?source=Sepolia\&dest=HashKey%20Sepolia\&token=HSK)
